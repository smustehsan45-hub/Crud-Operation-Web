🚀 MERN Stack Web Application

This is a full-stack web application where users can register, log in, and interact with the platform, while an admin has full control to manage users, contacts, and services.

📌 Features
👤 User Features
User Registration (Sign Up)
User Login (Authentication)
Secure access to personal data
🛠️ Admin Features
Admin Dashboard
Manage Users (Create, Read, Update, Delete)
Manage Contacts (CRUD Operations)
Full control over system data
📦 Services
Services are dynamically fetched from the database
Easily manageable and scalable
🧑‍💻 Tech Stack
Frontend
React.js
React Router
Axios
Backend
Node.js
Express.js
Database
MongoDB (Mongoose)
Authentication
JWT (JSON Web Tokens)
Password hashing (bcrypt)
